# 🌌 Wastelands – Sci-Fi Game Landing Page UI

A cinematic **game landing page UI concept** designed in **Figma**.
The design presents a mysterious alien world filled with glowing crystals, cosmic storms, and dangerous environments.

The goal of this project was to explore **immersive storytelling UI for game marketing websites**.
